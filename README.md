A simple explorer that receives a webpage as parameter to launch the game.

### Usage
./mgl.py [options] url

### Options
``-f`` o ``--fullscreen``: Fullscreen mode

``-t`` o ``--title``: Add a title to the window

``--width``: Redefine window size

``--height``: Redefine windoe height

### Example
./mgl.py -f google.com

### Requirements
- python-webkit
